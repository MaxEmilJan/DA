 Pins used for the connection between the Arduino Nano and the Adafruit Neopixel:
 D2        -->  Data Input;
 5V        -->  PWR;
 GND       -->  GND;
 
 Pins used for the I2C-connection between the NVIDIA Jetson and the Arduino:
 27 (SDA)  --> A4 (SDA);
 28 (SCL)  --> A5 (SCL);
 GND       --> GND
